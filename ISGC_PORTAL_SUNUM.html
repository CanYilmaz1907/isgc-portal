Kurumsal İş Sağlığı ve Güvenliği Yönetim Sistemi

Production seviyesinde, modüler ve ölçeklenebilir çözüm

1

Neden ISG-C Portal?

  • Tek platform – Tüm ISG süreçleriniz tek yerde
  • Otomatik bildirimler – E-posta ile takip ve hatırlatmalar
  • Raporlama – İstatistikler, grafikler, trend analizi
  • Denetim uyumu – Kontrol listeleri, arşivleme
  • Rol bazlı yetkilendirme – Güvenli erişim kontrolü
2

Ana Modüller

Kaza Yönetimi Uygunsuzluk Takibi Disiplin Logu Denetim & Kontrol Listeleri Denetim Arşivi Doküman Yönetimi Kaza Raporlama

Kaza Yönetimi

Dinamik formlar, kök neden analizi (5N1K), otomatik e-posta bildirimleri, diyagram oluşturma.

Uygunsuzluk Takibi

Proje bazlı takip, fotoğraf yükleme, tehlikeli sınıf tanımlama, otomatik hatırlatmalar.

Denetim

İç/dış denetimler, kontrol listesi yönetimi, otomatik skor hesaplama, HTML raporlar.

3

Teknoloji Altyapısı

Backend

Spring Boot 3.3 Java 17

JWT kimlik doğrulama, RBAC, JPA/Hibernate, Flyway, Swagger API, AOP audit logging, SMTP mail servisi.

Frontend

React 18 TypeScript Vite

Ant Design 5, ECharts grafikler, React Router, responsive tasarım.

Veritabanı

PostgreSQL 16 – güvenilir, ölçeklenebilir, kurumsal standartlara uyumlu.

5

Otomasyon & Entegrasyon

  • E-posta bildirimleri – Kaza, uygunsuzluk, denetim tamamlandığında otomatik bildirim
  • Periyodik raporlar – Abonelik sistemi ile düzenli kaza raporları
  • Hatırlatma e-postaları – Uygunsuzluk takibi için otomatik hatırlatmalar
  • PDF raporlar – Kaza, disiplin, denetim raporları otomatik oluşturulur
  • Swagger API – Tüm endpoint'ler dokümante, entegrasyon kolaylığı
6

Kurulum & Dağıtım

Gereksinimler: Java 17, Maven, Node.js 18+, PostgreSQL (veya Docker)

Hızlı Başlangıç

  • docker compose up -d db – Veritabanı
  • mvn spring-boot:run – Backend (port 8080)
  • npm run dev – Frontend (port 5173)

Production

JAR olarak paketlenebilir, Nginx ile reverse proxy, HTTPS destekli dağıtım.
